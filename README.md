# Kalorienverbrauch-Vorhersage bei Garmin

## Beschreibung
Diese Fallstudie untersucht den Einsatz von Machine Learning 
zur Vorhersage des individuellen Kalorienverbrauchs beim Sport 
am Beispiel des Unternehmens Garmin.
